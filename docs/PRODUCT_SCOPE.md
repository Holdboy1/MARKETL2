# Product Scope

## Produto

Plataforma white-label de marketplace para servidores de jogos online.

## Problema resolvido

Servidores de jogos operam economia e trade de forma improvisada, com baixa
visibilidade, pouca confianca de dados e dependencia excessiva de canais de
Discord ou processos manuais.

## Solucao

Uma plataforma SaaS multi-tenant que permite a cada servidor:

- publicar um marketplace com identidade propria
- consultar itens, precos e historico
- configurar alertas e notificacoes
- integrar Discord e APIs do proprio servidor
- operar com painel administrativo proprio

## MVP

### Publico

- home do marketplace por tenant
- listagem de itens
- filtros por item, moeda e faixa de preco
- historico basico de preco
- pagina de alertas

### Administrativo

- cadastro de tenant
- configuracao de nome, logo e moeda
- configuracao de integracoes
- visao basica de logs e saude operacional

### Integracao

- ingestao de itens via bot/Discord
- ingestao via API autenticada
- disparo de alertas por Discord

## Fora do MVP

- pagamentos integrados
- cobranca automatica
- marketplace cross-server
- analytics avancado com cohort/funnel
- recomendacao inteligente de precos

## Modelo comercial inicial

- setup de implantacao
- mensalidade recorrente
- customizacao premium opcional
- integracoes especiais cobradas a parte
